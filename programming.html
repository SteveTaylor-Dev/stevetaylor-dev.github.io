Modular Accelerometer Control (2017) - C#, Unity

A refactoring of the accelerometer-control project, with the focus being on smaller, specific behaviours per script, which can then be added to an object. More behaviour scripts have been added, and are built to be compatible with one another. This means that object behaviour can be built using modular scripts alone, with no coding required.

Accelerometer Control (2017) - C#, Unity

Created to test an idea involving using a smartphone accelerometer as input. Contains Game, Camera and Gravity controllers, each with optional behaviours utilizing the accelerometer as input (with keyboard alternatives available).

Multiplayer Scaling System (2016) - C#, Unity

One of the earliest examples of code I've written, this project was created after following Unity learning materials to create an arcade-style space shooter. Wanting to take the project much further than the materials covered, I added multiple systems to handle functionality such as; health management, adjustable weapon attributes, drop-in/drop-out multiplayer, difficulty/reward scaling (based on player amount), 'even rewards' system that equally distributes between players, and an object pooling system. More detail on these functionalities can be found on the GitHub repo linked below.
